Project Details

Main motive of this project is to develop a system to store, analyze, and visualize Twitter’s tweets
This project consisted of different phases to reach to the visualization part:

  • Phase 1: Collect Tweets using Twitter’s Streaming APIs and extracting all the hashtags and URLs in the tweets
  • Phase 2: Design and implement your ideas using Apache Spark
    • Step 1: Store the tweets in Spark SQL,
    • Step 2: Write interesting analytic queries to explore and understand the data
    • Step 3: Develop interesting visualizations (e.g., pie chart, heat map, bar graphs)
  • Phase 3: To present your project in the form of an attractive poster (website) and a demo of your software (video)

Language Visualization

This is a bubble chart visualization which shows the languages used to tweet, and the language with the most number of tweets has the biggest bubble.

Tweet Type Visualization

This is a bar graph visualization which shows the most type of the tweet in the dataset, which can be a comment,quote of a comment, retweet of a comment and retweet of a quote.

Gealocation Visualization

This is the Gealocation visualization, based on the location of the tweets in the dataset, it shows the different places from where the tweet was done.

Tweet Source Visualization

The number of different types of sources used for tweeting are represented through this Pie Chart.
